📚 Java 泛型语法全景总结

## 泛型类
这是最基础的形式，类型参数  定义在类名后面。

语法模板：
```java
public class 类名<T> {
    // 成员变量可以是 T
    private T data;

    // 构造器可以使用 T
    public 类名(T data) {
        this.data = data;
    }

    // 普通成员方法可以使用 T
    public T getData() {
        return data;
    }
}
```


关键点：
作用域： 在整个类内部有效。
实例化：使用时必须指定具体类型，如 new 类名("hello")。

## 泛型接口
接口也可以定义类型参数，实现类需要决定是保留泛型还是指定具体类型。

语法模板：
```java
// 定义泛型接口
public interface 接口名<T> {
    T get();
    void set(T t);
}

// 实现类方案 A：指定具体类型（最常用）
public class 实现类A implements 接口名<String> {
    public String get() { return "hello"; }
    public void set(String s) { }
}

// 实现类方案 B：保持泛型（传递泛型）
public class 实现类B<T> implements 接口名<T> {
    public T get() { return null; }
    public void set(T t) { }
}
```


## 泛型方法
这是您刚才关注的重点。泛型方法可以独立于类存在，类型参数定义在返回值之前。

语法模板：
```java
// 普通泛型方法
public <T> void 方法名(T t) { ... }

// 静态泛型方法（必须自己声明 <E>）
public static <E> void 静态方法名(E e) { ... }

// 带返回值的泛型方法
public <T> T 方法名(T t) { return t; }
```

关键点：
位置铁律： 必须在修饰符（如 public static）之后，返回值类型之前。
独立性：方法的 <T> 和类的 <T> 是两码事（除非方法在泛型类内部且没重新声明 <T>）

## 通配符（PECS）
用于方法参数中，增加灵活性。
| 写法 | 含义 | 场景 | 能做什么？ |
| :--- | :--- | :--- | :--- |
| `List<?>` | 任意类型列表 | 只读不写（除了 null） | 只能读取为 `Object` |
| `List<? extends T>` | T 或 T 的子类 | 生产者 (Producer) | 只能读取，读出的是 `T` |
| `List<? super T>` | T 或 T 的父类 | 消费者 (Consumer) | 只能写入，写入 `T` |



## ⚠️ 核心注意事项（避坑指南）

### 静态方法不能访问类级别的泛型
这是最高频的错误。
错误：class Box { public static void print(T t) {} }
原因：静态方法属于类，类加载时 T 还没确定。
解决：静态方法必须自己声明泛型，如 public static  void print(E e) {}。

### 泛型数组创建限制
错误：T[] arr = new T[10]; （编译报错）
原因：由于类型擦除，运行时不知道 T 是什么，无法分配内存。
解决：使用 List 代替，或者使用反射 Array.newInstance(clazz, size)。

### 基本类型不能作为泛型参数
错误：List list = new ArrayList();
解决：必须使用包装类，如 List。

类型擦除与 instanceof
错误：if (obj instanceof T) （编译报错）
原因：运行时 T 被擦除为 Object 或边界类型，无法判断。
解决：传入 Class clazz 参数，使用 clazz.isInstance(obj)。

### 泛型不具备多态性（不变性）
错误：List strs = new ArrayList(); List objs = strs; （编译报错）
原因：List 不是 List 的子类。
解决：使用通配符 List 或 List。

## 📌 综合代码示例

这是一个包含所有正确写法的完整模板：
```java
import java.util.ArrayList;
import java.util.List;

// 1. 泛型类
public class GenericTemplate<T> {
    
    private T data;

    public GenericTemplate(T data) {
        this.data = data;
    }

    // 2. 使用类的泛型 T
    public T getData() {
        return data;
    }

    // 3. 泛型方法（非静态）：可以使用类的 T
    public void printData() {
        System.out.println(data);
    }

    // 4. 泛型方法（独立泛型）：定义自己的 <E>
    public <E> void printGeneric(E e) {
        System.out.println(e);
    }

    // 5. 静态泛型方法：必须自己定义 <E>
    public static <E> void staticPrint(E e) {
        System.out.println("Static: " + e);
    }

    // 6. 使用 PECS 的方法
    // 从 source 读取 T (Producer -> extends)
    public void copyFrom(List<? extends T> source) {
        for (T item : source) {
            System.out.println(item);
        }
    }

    // 向 target 写入 T (Consumer -> super)
    public void copyTo(List<? super T> target) {
        target.add(data);
    }

    public static void main(String[] args) {
        // 测试
        GenericTemplate<String> template = new GenericTemplate<>("Hello");
        template.printData();
        template.printGeneric(123); // E 推断为 Integer
        GenericTemplate.staticPrint("World"); // 静态调用
    }
}
```
