产品背景
鸿蒙星空是围绕提升用户留存、增强用户对鸿蒙系统的熟悉度与使用粘性而推出的一项
元服务。该服务通过设置一系列轻量化的日常任务，引导用户在完成任务的过程中，逐步熟悉鸿蒙
系统的各项功能、操作逻辑，从而加深用户对系统的认知与使用习惯，最终实现提升用户活跃度、
延长使用时长、巩固用户留存的核心目标。
项目架构
端侧：搭载鸿蒙系统设备，包括华为手机、平板、笔记本等。
云测：用户成长旅程服务、成长旅程管理服务、鸿蒙花园服务、用户福利中心服务

使用的技术栈
spring springboot springMvc mybatisPlus mysql 分库分表 redis rabbitmq
